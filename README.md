apparat
==========

## About
apparat is a python based application launcher for linux


## Requirements
- python
- python-wxtools


## Plugins
### Search
- !a (amazon)
- !b (bandcamp)
- !g (google)
- !r (reddit)
- !s (soundcloud)
- !t (twitter)
- !v (vimeo)
- !w (wikipedia)
- !y (youtube)

Usage: 
```trigger + searchphrase```

### Other
- !l (locks the user session)
