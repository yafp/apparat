apparat
==========

## About
apparat is a python based application launcher for linux


## Requirements
- python2
- python-wxtools
- gtk


## Plugins
### Internet-Search

Usage: 
```trigger + searchphrase```

- !a (amazon)
- !b (bandcamp)
- !g (google)
- !r (reddit)
- !s (soundcloud)
- !t (twitter)
- !v (vimeo)
- !w (wikipedia)
- !y (youtube)



### Session-Lock
- !l (locks the user session)
